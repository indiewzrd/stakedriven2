StakeDriven is a community crowdfunding platform for web2/web3 startups. Using web3 fundable polls, your users stake on what to build next. At the end of a poll, funds from the winning options are transferred to the poll creator, while funds from non-winning options are returned to the voters.

Non-audited raw testnet version of the dapp can be found at https://stakedriven.vercel.app/
